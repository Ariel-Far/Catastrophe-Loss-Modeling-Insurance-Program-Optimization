Portfolio Project

Catastrophe Loss Modeling & Insurance Program Optimization

A catastrophe-risk modeling project built from a Runner-Up Aon-sponsored case competition submission, then rebuilt and extended into an interactive analytics tool. This project combines Monte Carlo simulation, tail-risk reasoning, and insurance program comparison into a clean, decision-focused showcase.
